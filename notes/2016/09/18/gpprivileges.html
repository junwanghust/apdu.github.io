GP Application Privileges Coding

2016-09-18

DAP Verification

Application is capable of verifying a DAP.

Mandated DAP Verification

Application is capable of and requires the verification of a DAP for all load operations.

CVM management

This privilege indicates that the Application being installed will have the ability to successfully invoke the setPin(), update(), blockState(), resetAndUnblockState() and setTryLimit() methods which will cause the value of the global PIN or variables pertaining to the management of the Global PIN to change.

Card Reset

This privilege indicates that the Application being installed for make selectable becomes the default selected application and has the privilege to modify historical bytes on one or more card interfaces.

Card Terminate

This privilege indicates that the Application being installed will have the ability to successfully invoke the terminateCardManager() or terminateCard() methods, which will cause the card to reject all commands besides the GET DATA command.

Card Lock

This privilege indicates that the Application being installed will have the ability to successfully invoke the lockCardManager() or lockCard() methods, which will cause the card to be locked.

Delegated Management:

Application is capable of Delegated Management.

Global Service

Application provides services to other Applications on the card.

Final Application

The only Application accessible in card Life Cycle State CARD_LOCKED and TERMINATED.

Global Registry

This privilege indicates that the Application being installed Application may access any entry in the GlobalPlatform Registry.

Global Lock

This privilege indicates that the Application being installed may lock or unlock any other application.

Global Delete

Application may delete any Card Content.

Token Verification

Application is capable of verifying a token for Delegated Card Content Management.

Authorized Management

Application is capable of Card Content Management.

Trusted Path

Application is a Trusted Path for interapplication communication.

Ciphered Load File Data Block

Mandated Ciphered Load File Data Block. Is capable of the ciphering the encrypted load file of confidential content management.

Receipt Generation

Application is capable of generating a receipt for Delegated Card Content Management.
